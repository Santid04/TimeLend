# Frontend

Este workspace contiene una aplicación `Next.js` mínima pero ordenada, con carpetas listas para crecer:

- `app`: rutas y layout principal
- `components`: piezas visuales reutilizables
- `lib`: configuración y utilidades del frontend
- `hooks`: hooks de React preparados para futuras integraciones
- `types`: contratos específicos del frontend

La UI actual es un demo scaffold. Está pensada para ser reemplazada o expandida por v0 en etapas posteriores.
